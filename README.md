# MahlineShop.Server

MahlineShop is a backend project written using .NET10

## Structure

Catalog: SQL + Vertical Slices.

Identity: JWT + Custom Security.

Basket: Redis + Distributed Caching.

Ordering: Event-Driven Architecture + Snapshot Pattern.

Infrastructure: Central Package Management, Docker, Serilog, and Scalar UI.
